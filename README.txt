This is a Port Scanner written in Python3. It will scan for the more popular ports and list thei respective protocols.
It'll work on any of the Linux Distributions.

Make sure Python3 and pip are installed.

The program must be run with root privileges if ping3 package hasn't been previously installed on your computer.

Syntax :

git clone https://github.com/hawks23/portscanner.git
cd portscanner
chmod +x scanner.py
sudo python3 scanner.py <ip_addressToScan>
